# ラストコカー・コカー
# 数を大きくするゲーム

## 実行環境の必要条件(多分)
* python >= 3.10
* pygame >= 2.1


## ゲームの概要
* 敵を攻撃して倒し続けるゲーム

## ゲームの遊び方
* こうかとんを縦横、または上下に動かして、敵を攻撃する。
* ふたつゲームがあるため、遊び方は異なる

## ゲームの実装
### 共通基本機能
* 2つのゲームの選択画面

### 分担追加機能
* コカーコカーを選択した際に別のpyファイルを開くようにする
* コカーコカーにおいて、こうかとんが壁を通るとアイテムを取得できるようにする 
* コカーコカーにおいて、敵を攻撃できるようにする

### メモ
* 代表者が更新する前の状態で共有してしまったため、代表者の内容と異なる。
* 今回は2つのゲームを制作した。１つ目はランダムな数字の枠を通過することで装備を強化するシューティングゲームである。
2つ目はランダムな数字の枠を通過することでこうかとんが増加するシューティングゲームである。



## ゲームの遊び方
* アローモードの場合： 矢印キーでこうかとん上下に操作し，2つうち
の片方の枠を通過して、こうかとんの装備を強化させる。
* 残機が0になったらゲームオーバーとなる
* ミリタリーモードの場合：矢印キーでこうかとん左右に操作し，2つうちの片方の枠を通過して、こうかとんの残機を増減させる。
* HPが0にになったらゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 2つのモードから選択してもらう
* 画面が自動に縦に移動する
* キャラクターを左右に移動することが出来る
* ランダムに生成された2つの数字の枠組みを上から下へ流す
* 生成された数字を潜ることにより数字が増減される
* 2つのうち、片方の数字の枠組みのみ取得できる

* 参考URL：[ホーム画面](https://note.com/mjay/n/nf53d9beea2b5)
*それぞれ追加

### 分担追加機能
*・ホーム画面機能（担当：羽地）：プレーヤーに任意のゲームを選択させるスタート画面に関するクラス
*・ミリタリーモード機能（担当：長澤、佐野）：こうかとんの残機、乱数枠、BOSSとの戦闘に関するクラス

### ・佐野担当
1. 2つまとめてとったら、左の方を採用する
2. 1個取ったら、もう片方が消える
3. こうかとんに変更ができる
4. 有利になるものを取ったら、笑顔になり、不利になるものを取ったら、悲しい表情になる

*・アローモード機能（担当：鹿股、高以良）：こうかとんの弓、乱数枠、敵との戦闘に関するクラス
* 調理機能（担当：巻）：無料スマホゲーム特有の広告を途中に挿入するクラス

* すべてのクラスに関係する関数は，クラスの外で定義してある
